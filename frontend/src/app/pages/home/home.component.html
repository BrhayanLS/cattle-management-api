Registro de Ventas

Registra las ventas de ganado y sus costos asociados para calcular las ganancias netas

Tablas Principales:
  • Cattle (Ganado): Almacena datos básicos del ganado, como su identificación, estado, fecha de nacimiento y propietario
  • Owner (Propietario): Contiene información del propietario del ganado, como nombre, apellido, contacto y credenciales
  • Sale (Venta): Registra las ventas de ganado, incluyendo fecha y detalles financieros como precio por kilo, báscula y camión
  • Sale_Cattle (Venta de Ganado): Relaciona ventas individuales de ganado con registros de ganado y venta.
  • Roles: Define roles de usuario como ADMIN, USER e INVITED
  • Vistas: Son consultas personalizadas que combinan datos de varias tablas de la base de datos, simplificando el acceso a información específica

Tecnologias Usadas

Autenticación y Autorización:
  • Peticion: Cuando envías una solicitud a la API, se verifica si tienes un "token" de sesión. Si no lo tienes, se inicia sesión y se crea un nuevo token utilizando un sistema llamado JWT (JSON Web Token)
  • Autorización: Después de obtener el token, se verifica su validez y se determina qué acciones puedes realizar según tu rol de usuario. Esta información está codificada dentro del token
  • Respuesta: Si todo está bien, se procesa tu solicitud y se te devuelve una respuesta con la información solicitada de la base de datos
  • Uso del token: El token de acceso debe incluirse en todas tus solicitudes a la API después de iniciar sesión. Este token es válido por un día y luego necesitarás iniciar sesión nuevamente
